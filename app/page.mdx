# Interactive React Hooks Quiz

반갑습니다. 저는 이현서이고, 이제 막 리액트를 2년 정도 사용해온 개발자입니다.

```js showLineNumbers
setName("John");
console.log(name); // "David" ..??
```

리액트를 처음 배웠을 때 위와 같은 오류(?) 때문에 한동안 인터넷을 뒤져봤던 기억이 납니다. 리액트는 분명 상태관리를 간편하게 해주지만, 원리를 제대로 이해하지 못하고 사용한다면 예상과 다른 동작을 하는 코드를 작성하게 될 수도 있습니다.

이 퀴즈는 마치 리액트를 막 시작한 사람들을 위한 길라잡이와 같습니다. 입문자 입장에서 쉽게 헷갈릴 수 있는 부분들을 짚기 위해 노력했습니다. 리액트를 오래 사용해오셨다면, 리액트에 대해 얼마나 잘 이해하고 있는지 테스트하는 용도로 추천드립니다.

이 퀴즈에서는 리액트의 기본 작동 방식, `useState`, `useEffect`와 같은 훅, 그 외 다양한 키워드들을 다룹니다.
